## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/intellix-2.0.git
   cd intellix-2.0
   ```

2. Install the requirements
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application
   ```bash
   streamlit run app.py
   ```
